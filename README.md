# web-utbk
# 📝 Aplikasi Simulasi UTBK (Tes Potensi Skolastik)

Aplikasi web sederhana berbasis **HTML, CSS, dan JavaScript** murni untuk mensimulasikan ujian UTBK SNBT secara mandiri. Proyek ini dibuat sebagai sarana belajar web development dan persiapan ujian masuk perguruan tinggi.

## 🚀 Fitur Utama
* ⏳ **Timer Otomatis**: Hitung mundur waktu ujian (5 menit).
* 🧭 **Grid Navigasi Nomor**: Memudahkan pengguna melompat antar-soal secara acak.
* 🟡 **Fitur Ragu-Ragu**: Menandai nomor dengan warna kuning layaknya aplikasi UTBK asli.
* 🟢 **Indikator Warna**: Abu-abu (belum diisi), Hijau (sudah diisi), Kuning (ragu-ragu).
* 📊 **Skor Otomatis**: Kalkulasi nilai akhir dengan skala 0-1000 setelah ujian selesai.

## 🛠️ Tech Stack yang Digunakan
* **HTML5** - Untuk struktur konten.
* **CSS3** - Untuk desain tata letak (*sidebar layout*).
* **JavaScript (Vanilla)** - Untuk logika mesin kuis dan penyimpanan data.

## 💻 Cara Menjalankan Proyek
1. Clone atau unduh repositori ini.
2. Buka folder proyek.
3. Klik dua kali pada file `index.html` untuk menjalankannya di browser kesayangan Anda.
